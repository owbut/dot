# BSPWM dotfiles
These are my dotfiles. Pretty simple, configuration paths are wack for a lot of programs because I change them, but it shouldn't effect you. Polybar configuration will be for my monitors and will be irrelevant to you, so tweak as you must.

## Programs used

-[hsetroot](https://github.com/himdel/hsetroot)
